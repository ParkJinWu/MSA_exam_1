## 🏁Goal

- MSA 아키텍처를 직접 구성
- Eureka, Ribbon 을 이용해서 분산처리 시스템을 구성
- MSA 환경에서 병목 현상이 발생 하여도 쉽게 오류가 나는 부분을 찾는다.
- Redis 를 활용한 캐싱 개념을 이해하고 필요한 곳에 활용
- Docker 의 개념을 이해하고 Git Push 만으로 서버에 자동으로 업데이트
